# 类型转换

## 一、数字与字符串之间的类型转换

### 1.str()函数—将数字转换成字符串

```python
>>> str(1000)
'1000'
>>> str(9.07)
'9.07'
```



### 2.`int()`—将字符串或浮点数转换成整数

```python
>>> int("1000")
1000
>>> int(9.07) # 浮点数转整数默认下取整
9
```



### 3.`float()`—将字符串或整数转换成浮点数

```python
>>> float("100")
100.0
>>> float("100.01")
100.01
>>> float(10)
10.0
```
